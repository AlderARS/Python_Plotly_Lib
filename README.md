# 🛒 Dashboard Interativo de E-commerce com Python, Dash e Plotly

## 🚀 Sobre o Projeto

Este projeto consiste na criação de um dashboard interativo para análise de dados de um e-commerce utilizando **Python**, **Dash** e **Plotly**.

A aplicação permite visualizar métricas de vendas e comportamento dos produtos de forma dinâmica, possibilitando filtrar os dados por temporada e explorar diferentes perspectivas através de múltiplos gráficos interativos.

O objetivo é transformar dados brutos em informações visuais que apoiem a tomada de decisão e a análise de desempenho dos produtos.

---

# 🎯 Objetivos

* Desenvolver uma aplicação web interativa para análise de dados.
* Explorar diferentes tipos de visualizações utilizando Plotly.
* Aplicar filtros dinâmicos para segmentação dos dados.
* Criar dashboards voltados para Business Intelligence (BI).
* Demonstrar o uso do Dash para construção de aplicações analíticas.

# 📈 Visualizações Desenvolvidas

## 📊 Histograma

**Objetivo:** Avaliar a distribuição das notas dos produtos.

Indicador analisado:

* Nota dos produtos

Insight:

* Identificação da aceitação geral dos produtos pelos clientes.

---

## 💰 Gráfico de Dispersão

**Objetivo:** Analisar a relação entre preço e desconto.

Indicadores analisados:

* Preço
* Desconto

Insight:

* Identificar padrões de precificação e estratégias promocionais.

---

## 🔥 Mapa de Calor

**Objetivo:** Visualizar a relação entre descontos e quantidade de avaliações.

Indicadores analisados:

* Desconto
* Número de Avaliações

Insight:

* Entender o impacto dos descontos no engajamento dos consumidores.

---

## 📦 Gráfico de Barras

**Objetivo:** Comparar o volume de vendas por categoria de gênero.

Indicadores analisados:

* Gênero
* Quantidade Vendida

Insight:

* Identificar categorias mais populares entre os consumidores.

---

## 🥧 Gráfico de Pizza

**Objetivo:** Demonstrar a participação de cada produto nas vendas totais.

Indicadores analisados:

* Produtos
* Quantidade Vendida

Insight:

* Avaliar quais produtos possuem maior representatividade nas vendas.

---

## 📉 Gráfico de Densidade

**Objetivo:** Analisar a distribuição dos preços dos produtos.

Indicadores analisados:

* Produto
* Preço

Insight:

* Identificar padrões de precificação entre os itens comercializados.
